## 新投稿 (6)

<h3 id="1">[1] ChipCraftBrain: Validation-First RTL Generation via Multi-Agent Orchestration</h3>
- **arXiv**: [2604.19856](https://arxiv.org/abs/2604.19856)
- **Authors**: Cagri Eryilmaz
- **Subjects**: cs.AR; cs.AI; cs.LG
- **Tags**: RTL Generation, LLM Agent, Multi-Agent System
- **Summary**: 本文提出ChipCraftBrain框架，结合符号神经推理与自适应多智能体编排实现自动化RTL生成。该方法在VerilogEval-Human基准上达到97.2%的pass@1，在NVIDIA CVDP基准上达到94.7%，显著优于现有方法。

<h3 id="2">[2] Efficient Page Migration in Hybrid Memory Systems</h3>
- **arXiv**: [2604.19932](https://arxiv.org/abs/2604.19932)
- **Authors**: Upasna, Venkata Kalyan Tavva
- **Subjects**: cs.AR
- **Tags**: Memory Architecture, Heterogeneous Computing
- **Summary**: 本文提出Duon，一种用于混合内存系统的高效页迁移方案，通过将更新后的映射信息直接存储在TLB和页表中，消除了页迁移后的TLB关机和缓存行失效开销。该方法相比现有技术实现了3.87%的IPC提升。

<h3 id="3">[3] Algorithm and Hardware Co-Design for Efficient Complex-Valued Uncertainty Estimation</h3>
- **arXiv**: [2604.19993](https://arxiv.org/abs/2604.19993)
- **Authors**: Zehuan Zhang, Mark Chen, He Li, Wayne Luk
- **Subjects**: cs.AR; cs.LG
- **Tags**: FPGA, Uncertainty Estimation, Hardware-Software Co-design
- **Venue**: DAC 2026
- **Code**: [code](https://github.com/zehuanzhang/BayesCVNN.git)
- **Summary**: 本文首次提出基于dropout的贝叶斯复值神经网络(BayesCVNNs)用于不确定性量化，并设计了FPGA加速器框架。优化后的加速器相比GPU实现达到约4.5x和13x的加速，功耗降低90%以上。

<h3 id="4">[4] EnergAIzer: Fast and Accurate GPU Power Estimation Framework for AI Workloads</h3>
- **arXiv**: [2604.20105](https://arxiv.org/abs/2604.20105)
- **Authors**: Kyungmi Lee, Zhiye Song, Eun Kyung Lee, Xin Zhang, Tamar Eilam, Anantha P. Chandrakasan
- **Subjects**: cs.AR
- **Tags**: Energy Efficiency, GPU Computing, AI Sustainability
- **Venue**: ISPASS 2026
- **Summary**: 本文提出EnergAIzer框架，通过轻量级方法预测硬件利用率输入，将AI工作负载的GPU功耗估计时间从数小时缩短到数秒。该方法在NVIDIA Ampere GPU上实现8%的功耗误差，并能以7%误差预测H100功耗。

<h3 id="5">[5] A Novel Low-Power Cache Architecture Based on 6-Transistor SRAM Cells</h3>
- **arXiv**: [2604.20176](https://arxiv.org/abs/2604.20176)
- **Authors**: Naser Khatti Dizabadi, Ceyda Elcin Kaya
- **Subjects**: cs.AR; cs.ET
- **Tags**: Low Power, Circuit Design, Memory Architecture
- **Summary**: 本文提出一种基于6T SRAM单元串联互连的低功耗缓存架构，利用堆叠效应抑制漏电流。该架构在不增加存储单元晶体管数量的情况下，有效降低了SRAM缓存的漏电功耗。

<h3 id="6">[6] Evaluating Computing Platforms for Sustainability: A Comparative Analysis of FPGAs against ASICs, GPUs, and CPUs</h3>
- **arXiv**: [2604.20638](https://arxiv.org/abs/2604.20638)
- **Authors**: Chetan Choppali Sudarshan, Aman Arora, Vidya A Chhabria
- **Subjects**: cs.AR; cs.ET
- **Tags**: FPGA, AI Sustainability, Energy Efficiency
- **Summary**: 本文介绍GreenFPGA工具，用于估算FPGA全生命周期的碳足迹，包括设计、制造、重配置、运行、处置等阶段。研究识别出FPGA在频繁变化、多样化工作负载和低产量应用场景下比ASIC、GPU和CPU更具可持续性。

<h3 id="9">[9] PVAC: A RowHammer Mitigation Architecture Exploiting Per-victim-row Counting</h3>

- **Subjects**: cs.CR; cs.AR

- **Tags**: Compute-in-Memory, Energy Efficiency, Matrix Multiplication
- **Venue**: ICM 2025
- **Summary**: 本文提出DISCA，一种使用压缩Bent-Pyramid格式的数字存内随机计算架构，继承了模拟计算的简洁性同时保持数字系统的可扩展性和可靠性。后仿真结果显示在180nm CMOS工艺下实现3.59 TOPS/W的能效。
